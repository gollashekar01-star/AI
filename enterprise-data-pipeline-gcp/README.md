# Cloud-Based ETL Pipeline (GCP Ready)

## Overview
This project demonstrates a modular ETL pipeline built using Python and PostgreSQL, designed for orchestration with Apache Airflow and cloud deployment (Google Cloud Platform).

## Architecture
Data Source (CSV) → Extract → Transform → Load → PostgreSQL

## Tech Stack
- Python
- PostgreSQL
- GCP (Cloud-ready design)
- Modular ETL design

## Future Enhancements
- Airflow orchestration
- Docker containerization
- GCP Cloud Storage integration
